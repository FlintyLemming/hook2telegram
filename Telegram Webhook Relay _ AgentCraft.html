Questions

Answers before you ship

If you need something custom, reach out—our team is happy to help.

What payload format does the webhook accept?

Send a POST request with the Content-Type set to application/json. The body must contain a message field. Any other fields are stored and shown in delivery logs.

Can I send messages to multiple Telegram chats?

Not yet. Right now each account can bind the service to a single Telegram chat ID. Multi-chat routing is on our roadmap—reach out if you need it.

Is message delivery reliable?

We queue and retry messages automatically. If Telegram is temporarily unavailable, we keep retrying with exponential backoff and show status updates in the dashboard.

How do I find my Telegram chat ID?

Open the dashboard and follow the built-in step-by-step guide—we walk you through starting a helper bot and copying the chat ID in under a minute.
